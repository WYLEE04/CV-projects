# Computer Vision Projects

This repository contains implementations of classical and deep learning-based computer vision algorithms for scene understanding, geometric vision, and image processing.

The goal of this project is to build fundamental vision systems from scratch and understand both low-level image processing and high-level semantic perception.

---

## 📌 Implemented Projects

### 1. Monocular Depth Estimation
- Transformer-based depth prediction
- Pixel-wise depth visualization
- Depth map normalization and colormap rendering
- GPU inference support

### 2. Object Detection (YOLO-based)
- Real-time object detection pipeline
- Bounding box visualization
- FastAPI deployment for inference

### 3. SIFT-based Panorama Stitching
- Feature extraction and matching
- Homography estimation (RANSAC)
- Image warping and panorama generation

### 4. Multi-band Blending
- Laplacian pyramid construction
- Multi-scale image blending
- Seamless compositing

### 5. Hough Circle Detection
- Edge detection (Canny)
- Hough Transform for circle detection
- Classical geometric feature extraction

---

## 🧠 Technical Highlights

- Classical Computer Vision (SIFT, Hough Transform, Homography)
- Deep Learning-based Vision Models (YOLO, Transformer-based Depth)
- Geometric Vision and Image Warping
- Multi-scale Image Processing
- API-based Model Deployment (FastAPI)
